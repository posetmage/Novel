# 戰鬥導覽
Entropy真是讓人又愛又恨的物理機制。  
戰鬥方式可能可以利用增加隊伍不同知識技術體系，大幅增加戰鬥區域的Entropy讓敵方魔王無法施放大規模技能，對於魔多血薄的魔王相當有效。  
反之對於魔少血厚的魔王建議隊伍只挑選少量技術體系，快速的發揮隊伍的特長連續施放相同技能  
嫌太麻煩的人，可以簡單地把自身練強用暴力方式解決，雖然我認為這個想法本身很蠢就是了。再說如果真的好練強，那對方不也是一樣很快地練強。  
  
*−−凱莉《戰鬥技術指南》*
