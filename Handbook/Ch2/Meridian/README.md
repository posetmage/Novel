### 小周天世界
氣在體內沿任、督二脈循環一週，即內氣從下丹田出發，經會陰，過肛門，沿脊椎督脈通尾閭、夾脊和玉枕三關，到頭頂泥丸，再由兩耳頰分道而下，會至舌尖(或至迎香，走鵲橋）

### 大周天世界
大周天是在小周天的基礎上更進一步的鍛煉與提升，將內氣延伸到四肢，是從頭頂百會到腳心的湧泉穴為止
