## 簡介
熱愛ACGN，目標是成為遊戲設計師，為此需要會everything，因為任何元素都可能讓遊戲增添內涵。  
所以平時會累積大量不同領域的知識儲備，ex:物理、經濟、演化、哲學、圍棋、投資、醫學、遊戲設計、重訓、音樂、繪畫、....，有空時會看大量的專業領域youtuber。  
閒暇之餘會將不同領域知識做整理或者聽各種類型音樂。
